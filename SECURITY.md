# Security

Usager puede acceder a sesiones locales, cookies autorizadas, archivos de configuración y elementos del Llavero según los proveedores habilitados.

No publiques cookies, tokens, claves de API, archivos de configuración ni reportes de diagnóstico sin revisar en un issue. Para reportar una vulnerabilidad, contactá a Leandro Riviello mediante [leandroriviello.com](https://www.leandroriviello.com).
